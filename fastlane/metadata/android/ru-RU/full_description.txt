GitNex — это бесплатный/платный Android-клиент с открытым исходным кодом для инструментов управления репозиториями Git — Forgejo и Gitea.

# Функции

- Поддержка нескольких учетных записей
- Браузер файлов и каталогов
- Просмотрщик файлов
- Создайте файл/проблему/запрос на слияние/вики/milestone/выпуски/ярлыки
- Список запросов на включение
- Список репозиториев
- Список организаций
- Список проблем
- Список меток
- Список контрольных точек
- Список выпусков
- Страницы вики
- Автономный (оффлайн) режим
- Исследуйте репозитории/проблемы/организации/пользователей
- Просмотр профиля
- Поддержка Markdown
- Поддержка Emoji
- Обширные Настройки
- Поддержка OTP для 2FA
- Уведомления
- Черновики
- Коммиты репозитория
- Поддержка самоподписанных сертификатов
- Темы
- и другое...
