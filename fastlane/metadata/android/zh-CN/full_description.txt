GitNex 是 Git 存储库管理工具 Forgejo 和 Gitea 的免费/付费的开源 Android 客户端。

# 功能

- 多账户支持
- 文件及目录浏览器
- 文件查看器
- 创建文件/议题/合并请求/维基/里程碑/发布/标签
- 合并请求列表
- 仓库列表
- 组织列表
- 问题列表
- 标签列表
- 里程碑列表
- 发行列表
- 维基页面
- 浏览代码仓库/issues/组织/用户
- 资料视图
- Markdown 支持
- Emoji 支持
- 扩展设置
- 通知
- 仓库提交
- 自签名证书支持
- 主题
- & 更多...
